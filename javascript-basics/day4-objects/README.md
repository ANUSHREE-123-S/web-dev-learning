# Day 4 - Objects, Destructuring, Spread

## Objects
- Store data in key-value pairs

Example:
const user = { name: "Anu", age: 20 };

## Destructuring
- Extract values easily

Syntax:
const { name, age } = user;

## Spread Operator
- Copy or update objects

Example:
const newUser = { ...user, age: 21 };

## Mistake
- Directly modifying original object ❌
